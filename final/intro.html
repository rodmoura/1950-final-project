Rationale

Here is some of the major points we discussed when thinking about how the Comp 1950 website would showcase a better layout and userflow towards the students. We went through the the aspects we liked and disliked when browsing the website to narrow down the main changes.

General

  • A new, better colour scheme focused on learning to help clearly define the different elements. The current colours are Red, Grey and White. Studies have proven that soft blues and greens are better suited for learning and education.
  • Improving the search functionality of the website for quick reference
  • Improve the SEO
  • HTML5 and CSS3 elements
  • Added jQuery functions
  • Use SSI footer and header to showcase the use of Server Side Includes
  • Added Print style

Landing/Index Page

  • Content is visible, the student will know exactly what lecture to click on, no more finding the link in the sidebar
  • More visible, user-friendly lecture links with a graphic to entice the user
  • Sticky Navigation: for ease of use, less waste of space per screen

Lectures pages

  • Clear concise difference between code examples and lecture material
  • Exercises for each lecture on a different page to allow the user to focus solely on that exercise
  • Establish a style for content boxes and code content
  • Easy to read interface with simplified lecture menu
  • Presntation style for overhead projectors

Exercises

  • All exercises now listed on a separate page from the lecture, making it easier to access without scrolling through the lecture notes

Colour Scheme

As stated above we went with set of colours for the header, footer, content headers and navigation. At the same time keeping the white and greys with black text. The will bring the students eyes to the right areas at the right time without seeming cluttered or overwhelming.









Fonts

We simplified the fonts into two easy to use and readable googlefonts:

  • Montserrat - for all the headers and titles
  • Roboto - for all the content
